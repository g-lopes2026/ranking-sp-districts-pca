# ranking-sp-districts-pca
  📍 Ranking de Performance dos Distritos de São Paulo
  
  Análise Espacial Interativa com PCA (Análise de Componentes Principais)

    Visão Geral do Projeto
  Este projeto transforma dados brutos (socioeconômicos e de performance) dos distritos de São Paulo em um Score de Ranking estruturado. Ao aplicar a técnica estatística PCA, reduzi múltiplas variáveis em uma única métrica de desempenho, permitindo uma comparação clara entre o que é "bom" e "ruim" em toda a cidade.

    Principais Etapas Técnicas
  1- Tratamento de Dados: Limpeza e normalização dos dados por distrito.
  
  2- Modelagem Estatística: Uso de PCA para atribuir pesos aos fatores de forma objetiva, eliminando subjetividades no ranking.
  
  3- Visualização Interativa: Mapa coroplético dinâmico construído com Geopandas e Folium (Mapclassify).
  
  4- Camadas de Mapa: Opções de visualização em Terreno, Claro e Escuro para facilitar a análise espacial.
